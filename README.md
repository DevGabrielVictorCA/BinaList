# 🚀 Bina List — Produtividade sem distrações.

A maioria das to-do lists falha por um motivo simples:

> Elas ajudam você a organizar… mas não a agir.

A **Bina List** foi criada para resolver exatamente isso.

---

## 🌐 Live Demo

Acesse a aplicação:

👉 [Acessar Bina List](https://www.binalist.online/)

---

## ⚡ O problema

Você já passou por isso:

- Uma lista cheia de tarefas  
- Várias prioridades competindo  
- Nenhuma clareza do que fazer agora  

Resultado?

> Você passa mais tempo gerenciando tarefas do que executando.

---

## 💡 A virada de chave

A Bina List elimina a indecisão.

Ao invés de te mostrar tudo, ela te mostra **o que importa agora**:

👉 **uma única próxima ação, clara e executável**

Sem ruído. Sem excesso. Sem fricção.

---

## 🎯 Como funciona

- Você cria tarefas rapidamente  
- Define prioridades simples  
- O sistema faz o resto  

Automaticamente, a aplicação:

- Ordena o que realmente importa  
- Ignora o que não é prioridade  
- Destaca sua próxima ação  

> Você não precisa decidir. Só executar.

---

## 🔥 Por que isso é diferente?

Porque não é mais uma to-do list.

É um sistema projetado com um único objetivo:

> **aumentar sua capacidade de agir agora**

- Menos decisões  
- Menos distração  
- Mais progresso real  

---

## 🧠 Filosofia do produto

Cada detalhe da Bina List segue uma regra simples:

> Se não ajuda o usuário a agir, não entra.

Isso resulta em uma experiência:

- Direta  
- Intuitiva  
- Sem sobrecarga mental  

---

## ⚙️ Funcionalidades principais

- Criação rápida de tarefas (Enter)  
- Sistema simples de prioridades  
- Destaque automático da próxima ação  
- Marcação de tarefas concluídas  
- Indicador de progresso em tempo real

## 💻 Como rodar o projeto

Clone o repositório:

```bash
git clone https://github.com/DevGabrielVictorCA/BinaList.git
```

Acesse a pasta:

```bash
cd BinaList
```

Abra o arquivo index.html no navegador ou utilize uma extensão como Live Server.
